# ARENA CRAWLER

Dungeon Shooter cenital basico para integrarlo con otras cosas

Mejor explicado en el cronograma creado por Claude

Esta porqueria se sostiene a punta de Claude

## REQUERIMIENTOS

- NodeJS + NPM
Para comprobar que este funcionando, se ejecuta
```console
node --version
npm --version 
```
Si sale error, te toca instalar algo de eso
- Algo algo Arduino. No se, no es mi departamento

## INSTALACION

```console
git clone https://github.com/ArielParedesLozada/arena-crawler.git
npm install
npm run dev
```

## EJECUCION

```console
npm run dev
```
Y mira en el navegador en: `localhost:5173`
